----SETUP----

--Server--
*Install Zerotier One
	*For Chrome OS install via google play store
*Connect to network ID : 65228d8d6d2afb9d
*Contact me to be authenticated on the vpn network

--Game--
*For Chrome OS:
	*Activate linux environment in settings
	*Install git by executing "sudo apt-get install git-all" in terminal
	*Download both linux scripts from github.com/bendo64/endostellar_online/tree/main/Linux scripts and paste them into your Linux folder in file system
	*Execute the following commands in the terminal:
		sudo chmod +x eso_setup.sh
		sudo chmod +x eso_update.sh
	*Execute "sudo ./eso_setup.sh" to install eso
	*Execute "sudo ./eso_update.sh" to update eso
	
	
*For Windows:
	*Copy .exe file
	
	
----STARTING GAME----
*Boot up Zerotier and connect to the endostellar.net vpn

*For Chrome OS:
	*Open up the terminal
	*Start the game using "./run_eso.sh" 
*For Windows:
	*Start .exe file
